# fish-depth-viewer
Web-based recreation of the FishBase “Fish at Depths Viewer,” simulating wavelength-dependent light attenuation and pupil compensation to show how fish colors change with depth.


Fish at Depth (Web)

This project is a browser-based recreation of the FishBase “Fish at Depths Viewer,”
which simulates how fish colors change with increasing water depth due to
wavelength-dependent light attenuation and pupil dilation.

The original concept and Java implementation were developed by:

Sherman R. Lai (Syracuse University)
Line B. Christensen (University of British Columbia)

The method and description are based on:
Lai, S.R. and Christensen, L.B. “The Fish at Depths Viewer,” created for FishBase
(www.fishbase.org).

This implementation is an independent JavaScript version intended for modern
web browsers.
